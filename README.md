# E-Commerce Website

A simple, secure e-commerce platform built with PHP, MySQL, and Bootstrap. This project allows users to browse products, add them to a cart, and complete purchases via integrated payment gateways. It's designed for small businesses or as a learning tool for web development.

## Features

- **User Authentication**: Secure login/signup with password hashing and session management.
- **Product Catalog**: Dynamic display of products with categories, search, and filters.
- **Shopping Cart**: Add/remove items with session-based storage and AJAX updates.
- **Checkout & Payments**: Integration with Stripe/PayPal for secure transactions.
- **Admin Panel**: Manage products, orders, and users (requires admin role).
- **Responsive Design**: Mobile-friendly UI using Bootstrap.

## Technologies Used

- **Backend**: PHP 7.4+, MySQL
- **Frontend**: HTML5, CSS3, JavaScript (jQuery for AJAX)
- **Framework**: Pure PHP
- **Tools**: XAMPP for local dev
